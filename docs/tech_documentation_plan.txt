3. TECHNICAL SPECIFICATIONS AND DATA
-------------------------------------
Data-analysis:
- Regions: DG, SUB, CA1, CA2, CA3 for left and right side of hippocampus, 
and combined them for left hippocampus and right hippocampus, 12 regions for each brain scan.
- Segmentation: HSF, a pipeline segmenting said DG, SUB, CA1, CA2, CA3 regions using deepLearning models from an T1w MRI scans,
(might add a parameter to choose how many models hsf uses).
- Pipeline: Pipeline to run in 1 command with Potential Advanced optional parameters. Input is a bids folder stucture 
and output is a CSV filled with features, like volume and curvature metrics for each region for each t1w brainscan.
- Sources: HSF library (https://github.com/clementpoiret/HSF) citation guide at the bottom. For papers an automated, 
geometry-based method for hippocampal shape and thickness analysis (https://www.sciencedirect.com/science/article/pii/S1053811923003336?via%3Dihub) 

Calculation and Output:
- Calculate volume and surface area/volume from segmentations
- Create 3D surface mesh .vtk file from segmentations
- Calculate the curvature of the .vtk mesh from selected points.
- Statistics: Median, percentiles, standard deviation, etc.
Output format: CSV file, where each patient is its own row.

CSV columns:
[ Patient ID | Session ID | Volume (whole) | SA/Vol (whole) | SPHARM Stats (whole) | Subfield stats... ]
